# Packets and Frames

## What is a Packet?

A packet is a small unit of data that is sent over a network.

Data is broken into packets before being transmitted.

## What is a Frame?

A frame is a packet at the Data Link layer.

It contains:
- Source MAC address  
- Destination MAC address  
- Data  

## Key Idea

- Packet → Layer 3 (Network layer)  
- Frame → Layer 2 (Data Link layer)  

## What I Learned

- Data is split into smaller pieces (packets)  
- Frames are used in local network communication  
- Different layers use different formats
