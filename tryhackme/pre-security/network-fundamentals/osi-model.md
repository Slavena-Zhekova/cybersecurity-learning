# OSI Model

## What is the OSI Model?

The OSI (Open Systems Interconnection) model is a framework used to understand how data travels through a network.

It consists of 7 layers.

## The 7 Layers

1. Physical  
2. Data Link  
3. Network  
4. Transport  
5. Session  
6. Presentation  
7. Application  

## What Each Layer Does

- Physical → Transmits raw bits  
- Data Link → Handles MAC addresses  
- Network → Routing (IP addresses)  
- Transport → Ensures data delivery (TCP/UDP)  
- Session → Manages sessions  
- Presentation → Data formatting  
- Application → User interaction  

## What I Learned

- Data moves through layers  
- Each layer has a specific role  
- OSI helps understand how networks work
